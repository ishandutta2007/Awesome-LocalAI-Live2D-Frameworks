## 🌟 Why Local AI + Live2D?

In 2026, the intersection of **Local LLMs** (like Ollama, Llama 3, Mistral) and **Live2D Cubism** technology has revolutionized how we interact with digital characters. This repository tracks the best tools to build:
- 🦊 **AI VTubers**: Fully autonomous streamers with emotional lip-sync and personality.
- 💬 **Desktop Companions**: Private AI assistants with high-quality 2D animations.
- 🛡️ **Privacy-First Interaction**: No cloud APIs, no data leaks—everything stays on your machine.
- 🎮 **Immersive Roleplay**: Advanced character memory and interactive motion.

## 📦 Open-Source GitHub Projects

### 🏗️ Dedicated Local AI + Live2D Frameworks

- **[Open-LLM-VTuber](https://github.com/Open-LLM-VTuber/Open-LLM-VTuber)** 🤖  
  Leading open-source framework combining local LLMs (Ollama) with Live2D for real-time VTuber avatars with emotional expression and lip-sync.
  
- **[LLM-Desktop-Assistant](https://github.com/search?q=LLM-Desktop-Assistant)** 🖥️  
  Open-source desktop AI companion with Live2D avatar support, voice interaction, and full local execution.

- **[Handcrafted Persona Engine](https://github.com/search?q=persona+engine+live2d)** 🧠  
  Framework for creating deeply customizable AI personas with Live2D integration and local LLM reasoning.

- **[Live2D + Ollama Stacks](https://github.com/search?q=live2d+ollama)** 🔗  
  Numerous community repositories combining Live2D Cubism SDK with Ollama for real-time animated characters.

- **[SillyTavern + Live2D Extensions](https://github.com/SillyTavern/SillyTavern)** 🎭  
  Popular open-source frontend for character-driven AI chat with excellent Live2D and motion support.

- **[VTube Studio API Integrations](https://github.com/search?q=vTube+studio+llm)** 📡  
  Open-source bridges connecting local LLMs to VTube Studio for powerful avatar control and expressions.

- **[AI-VTuber](https://github.com/search?q=AI-VTuber+local)** 🎬  
  Collection of projects for building fully local AI VTubers with Live2D, voice synthesis, and personality engines.

- **[Character AI Local](https://github.com/search?q=local+character+ai+live2d)** 👤  
  Open-source alternatives to Character.AI with strong Live2D and local LLM integration.

- **[MikuLive2D](https://github.com/search?q=miku+live2d+llm)** 🌸  
  and similar anime-style avatar frameworks for enthusiasts.

### 🧩 Additional Strong Open-Source Options

- **Live2D Cubism SDK** (official free SDK) + community LLM wrappers.
- **[Ollama + WebSocket Live2D Bridges](https://github.com/search?q=ollama+live2d)** — Many real-time integration projects.
- **[Rhasspy / Mycroft]** voice assistants with Live2D frontends.
- **[Godot Engine + Live2D + LLM plugins](https://github.com/search?q=godot+live2d)** — For building custom 2D animated AI worlds.
- **Unity + Live2D + local LLM** community solutions.
- **Persona Engine frameworks** for memory, emotion, and long-term character consistency.
